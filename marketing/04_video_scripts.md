## 📋 Production Notes

| รายการ | รายละเอียด |
|--------|-----------|
| **กล้อง** | ไม่ต้องถ่ายหน้า — ใช้ screen recording + animation |
| **เสียง** | Thai voiceover (ผู้ชาย โทนมั่นใจ ไม่แก่เกิน) |
| **BGM** | เพลง lo-fi/chill ไม่ดังกว่าเสียงพูด |
| **Subtitle** | ใส่ทุกวิดีโอ (80% ดูไม่เปิดเสียง) |
| **Format** | 9:16 (TikTok/Reels) + 16:9 (YouTube) |
| **Text** | ตัวหนังสือใหญ่ สีขาวบนพื้นดำ อ่านง่ายบนมือถือ |
| **CTA** | แสดงลิงก์/QR code ใน 5 วินาทีสุดท้าย |
