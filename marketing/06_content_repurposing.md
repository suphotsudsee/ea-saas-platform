# 06 — Content Repurposing (6 ช่องทาง)
## TradeCandle v11 — ริโพสต์/ปรับแต่งสำหรับแต่ละแพลตฟอร์ม

---

## 🔄 กลยุทธ์: 1 Core Content → 6 Channels

หลักการ: สร้างเนื้อหาหลัก 1 ชิ้น → ปรับ format/โทน/ความยาว สำหรับ 6 ช่องทาง

---

## 📱 Channel 1: Facebook Page/Group

**โทน:** เป็นกันเอง + ให้ความรู้ + มี CTA ชัด  
**ความยาว:** 150-300 คำ + รูป/วิดีโอ

### การปรับจาก Core Content:
| Core Content | Facebook Version |
|-------------|-----------------|
| 3-Wave อธิบาย | โพสต์ยาว + รูป infographic + คอมเมนต์ตอบคำถาม |
| Video 60s | ลงเต็ม +  caption สรุปจุดสำคัญ |
| Blog สอน PA/SMC | โพสต์ snippet + ลิงก์อ่านต่อ |
| Review ผู้ใช้ | Screenshot + review card + คอมเมนต์ tag คน |

### ตัวอย่างโพสต์:
```
📊 ทำไม EA ทั่วไปขายทองแล้วขาดทุน?

เพราะรอ TP เต็ม 100% → ราคากลับ → กำไรหาย

TradeCandle v11 ใช้ 3-Wave Cashout:
✅ Wave 1: ปิด 25% TP → กำไรแรกล็อค
✅ Wave 2: ปิด 50% TP → กำไรเพิ่ม
✅ Wave 3: Trailing → ไล่เทรนด์ไม่จำกัด

ใครเคยเป็น? คอมเมนต์เลย 👇
```

## 📱 Channel 2: TikTok / Instagram Reels

**โทน:** สนุก + ดึงดูด + เร็ว  
**ความยาว:** 15-30 วินาที (สั้นที่สุดดีสุด)

### การปรับจาก Core Content:
| Core Content | TikTok Version |
|-------------|---------------|
| 3-Wave อธิบาย | 15s แอนิเมชั่น คลื่น 3 ชั้น + text overlay |
| Video 60s | ตัดแบ่งเป็น 2 คลิป 30s (Part 1/2) |
| Blog PA/SMC | 30s สอน 1 filter ต่อคลิป (6 ตอน) |
| Review ผู้ใช้ | 15s green screenshot + text "ตื่นมาเห็นกำไร" |
| FB โพสต์ยาว | Quote ประโยคเด็ด + กราพเคลื่อนไหว |

### รูปแบบยอดนิยม:
1. **Screen recording** MT5 + text overlay
2. **Before/After** เทรดเอง → ใช้ EA
3. **Countdown** โปรส่วนลดจะหมด
4. **Myth vs Fact** EA ขาดทุน = false
