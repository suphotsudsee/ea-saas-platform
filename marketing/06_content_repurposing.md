## 🔄 Repurposing MethodContent 1 ชิ้น → 6 Channel

**Principles:** Content Creationหลัก 1 ชิ้น (Core Piece) Alreadyดัดแปลงเป็น 6 FormatFor 6 Channel

## 🎯 Example Core Piece

**Contentหลัก:** Articles "6 Smart Money Filters At TradeCandle v12 UseAnalyzePointEnterGold"

## 📱 ChannelAt 1: Facebook

### Format: PostStyle Carousel + Messageสรุป

**Repurposing Method:**
1. แบ่งArticlesเป็น 6 การ์ด (1 Filterต่อ 1 การ์ด)
2. แต่ละการ์ด: NameFilter + Description 2 บรรทัด + ภาพChartประกอบ
3. PostMessage: สรุปภาพรวม + CTA

**ExamplePost:**
```
🧠 6 Filter Smart Money At TradeCandle v12 UseAnalyzeGold

All 6 ต้องผ่านReadyTogether ถึงจะOpenStatus!
← เลื่อนViewการ์ดแต่ละFilter →

อยากรู้ลึกกว่าThis? ReadArticlesเต็ม: tradecandle.ai/blog/smart-money-filters

#TradeCandle #SmartMoney #XAUUSD
```

**Sizeภาพ:** 1080x1080 px (สี่เหลี่ยมจัตุรัส)
**สี:** พื้นBlack + ItemcharactersGold + ไฮไลท์เขียว

## 💬 ChannelAt 2: Line OA

### Format: MessageBroadcast + Link + ภาพ 1 ใบ

**Repurposing Method:**
1. สรุปArticlesเป็น bullet points Shortสุด (5-8 บรรทัด)
2. ใส่LinkReadMore
3. แนบภาพอินโฟChartิก 1 ใบ

## 🎵 ChannelAt 3: TikTok

### Format: clipShort 30 seconds (แนวตั้ง 9:16)

**Repurposing Method:**
1. CreateclipfromPointน่าสนใจAtสุดของArticles
2. ShowFilterStyle flash card (1 secondsต่อ 1 Filter)
3. ภาพChartเคลื่อนไหว + MessageUpทีละอัน
4. CTA: "Linkใน Bio"

**Scriptโครง:**
```
[0-3s] Hook: "6 FilterAtสถาบันUse เราดึงมาใส่ EA"
[3-18s] Flash cards: 6 Filter 3 secondsต่ออัน
[18-24s] ภาพChart: AllFilterผ่าน → SignalsEnter
[24-30s] CTA: "ลองUseFree → Linkใน Bio"
```

**Sizeวิดีโอ:** 1080x1920 px (9:16)
**Format:** MP4, 30fps
**Sound:** Musicประกอบ trendy + Soundเอฟเฟกต์ "ting" Allครั้งAtUpFilterใหม่

## 📺 ChannelAt 4: YouTube

### Format: วิดีโอยาว 5-8 minutes (แนวนอน 16:9)

**Repurposing Method:**
1. ExpandArticlesเป็นScriptวิดีโอ
2. AddExampleChartจริงfrom MT5
3. Screen recording Show Backtest
4. Addส่วน Q&A ท้ายclip

**โครงCreateวิดีโอ:**
```
0:00-0:30 Hook: "ทำไม EA GeneralGold TradingNoได้Performance?"
0:30-1:00 แนะนำ Smart Money Concepts
1:00-6:00 อธิบาย 6 Filterทีละอัน (45 secondsต่อFilter)
6:00-7:00 Show Backtest: AllFilterทำงานร่วมTogether
7:00-8:00 Q&A + CTA
```

**Sizeวิดีโอ:** 1920x1080 px (16:9)
**SEO:** Nameclip + Description + Tagsใส่ keyword LanguageThai
**Thumbnail:** ภาพสีGold พื้นBlack Itemหนังสือใหญ่ "6 Smart Money Filters"
