## 📢 Ad 1: Pain-Point Hook (โฟกัสปัญหา)

### รูปแบบ: Image Carousel (3 รูป)

**หัวข้อแคมเปญ:** "กำไรหายไประหว่างทาง?"

**Copy:**
```
😢 เคยเป็นไหม?

เปิด BUY ทองไป ราคาขึ้นไป +$50...
แต่พอรอ TP เต็ม → ราคากลับ → กำไรหายหมด
กลายเป็นขาดทุนแทน 😤

💡 TradeCandle v11 แก้ด้วย 3-Wave Cashout:
✅ Wave 1: ปิดกำไรที่ 25% TP → ล็อคกำไรแรก
✅ Wave 2: ปิดกำไรที่ 50% TP → กำไรเพิ่ม
✅ Wave 3: Trailing Stop → ไล่ตามเทรนด์

ผล: ยังไงก็ได้กำไรบางส่วน ไม่รอเต็ม 100%

🔥 เริ่มต้น 990 บาท/เดือน → ลิงก์ในคอมเมนต์
```

**Carousel รูป 3 ภาพ:**
1. กราฟทองขึ้นแล้วกลับ (เสียใจ) → ข้อความ "เคยเป็นไหม?"
2. แผนภาพ 3-Wave Cashout → ข้อความ "วิธีแก้: ปิดเป็นคลื่น"
3. Dashboard SaaS → ข้อความ "ดูผลจากมือถือ"

**Targeting:**
- อายุ: 28-50
- ความสนใจ: ทองคำ, Forex Trading, MT4, MT5, เทรด
- ประเทศ: ไทย
- ภาษา: ไทย

## 📢 Ad 2: Social Proof Hook (โฟกัสผลลัพธ์)

### รูปแบบ: Video Ad (30s)

**หัวข้อแคมเปญ:** "ทำไมเทรดเดอร์ทองเปลี่ยนมาใช้ AI"

**Copy:**
```
📊 เลขไม่โกหก

TradeCandle v11 บน XAUUSD M5:
✅ 3-Wave Cashout: ปิดกำไรเป็น 3 รอบ
✅ 6 Smart Money Filters: อ่านทรงสตรัคเจอร์อัตโนมัติ
✅ ATR Dynamic SL/TP: ไม่ต้องตั้งเอง
✅ Dashboard ดูผล real-time จากมือถือ

"ตื่นมา เปิด MT5 เห็นกำไร ไม่ต้องนั่งดูจอ"

🎯 ทดลองใช้ 7 วัน → ลิงก์ในคอมเมนต์
```

**วิดีโอสคริปต์:**
- วินาที 0-5: โลโก้ + "เทรดทองด้วย AI"
- วินาที 5-15: ตัดฉาก MT5 แสดง EA ทำงาน (เปิด/ปิดออเดอร์อัตโนมัติ)
- วินาที 15-25: แสดง Dashboard SaaS บนมือถือ
- วินาที 25-30: CTA "ทดลอง 7 วัน" + เว็บ URL

**Targeting:**
- อายุ: 30-50
- ความสนใจ: Algorithmic Trading, Gold Investment, Swing Trading
- Lookalike: คนที่เคยคลิก ad ก่อน
